# Port Scanner

# 

# A clean, fast, and multi-threaded TCP port scanner built in Python.

# 

# !\[License](https://img.shields.io/badge/License-MIT-green)

# 

# \# Overview:

# This project is a lightweight Nmap-like port scanner designed for educational purposes. It demonstrates key concepts in network programming, concurrency, and building professional CLI tools.

# 

# \# Features:

# \- Beautiful ASCII banner using `pyfiglet`

# \- Multi-threaded scanning for high performance

# \- Customizable port range and thread count

# \- Clean output with service name detection

# \- Robust error handling (KeyboardInterrupt, hostname resolution, etc.)

# \- Professional CLI using `argparse`

# 

# \# Tech Stack:

# \- Python 3

# \- `socket` – Low-level networking

# \- `threading` – Concurrency

# \- `argparse` – Command-line interface

# \- `pyfiglet` – ASCII art banner
